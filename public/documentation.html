Description URL Request Response
Allow Users to register. /users POST NONE
Allow Users to add movies to their favorites. /users/:id/:movieTitle POST A message stating movie has been added to favorites.
Return a List of movies. /movies GET A JSON object with all the movies.
Return data about a single movie. /movies/:title GET A JSON object with a single movie.
Return data about a movie by genre. /movies/genre/:genreName GET A JSON object about a movies genre.
Return data about a movies director. /movies/director/:directorName GET A JSON object about the movies director.
Allow users to update their user info. /users/:id PUT A JSON object with updated user info.
Allow Users to remove a movie from their favorite list. /users/:id/:movieTitle DELETE A Message stating movie has been removed from favorites.
Allow Users to unregister. /users/:id DELETE A message stating user has been removed.
